We are excited to add a new **site search** to help clients find information faster across the Twilight Psychology website and blog. Instead of browsing through multiple menus, you can now open search and go directly to the page you need. 

The search icon is available in the top header so you can open search without opening the menu first. Search is intended to reduce cognitive load and potential overstimulation for neurodiverse users.

## What search can help you find

The new search can help you quickly reach content such as:

- service pages for **therapy**, **medication management**, and **psychological evaluations**
- specialty pages such as **IEEs**, **guardianship evaluations**, and **psychoeducational evaluations**
- client resources like **fees**, **insurance information**, **new client guidance**, and the **masking inventory**
- blog posts and practice updates

## Good searches to try

If you are not sure where to start, try simple phrases such as:

- `ADHD testing`
- `Autism testing`
- `therapy`
- `insurance`
- `fees`
- `new client`
- `masking`

Short, specific terms usually make it easier to spot the page you want.

## When search is especially useful

Search is a helpful shortcut when you already know the topic you need, 
want to compare a few related pages quickly, or if you 
are trying to find a blog post, insurance update, or client resource without remembering where it lives in the navigation. If you prefer browsing, the main navigation is still available.
